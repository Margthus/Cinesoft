# CineSoft

Modern movie and TV show explorer built with Electron, React, and Vite.

## Features
- TMDB API Integration
- Netflix-style UI
- Dual Language Support (TR/EN)
- Neon Mint (#00ffcc) Aesthetic

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Configuration
Go to the **Settings** section in the app to:
- Enter your TMDB API Key.
- Change the application language.
